2. Welche Daten wir erheben

Accountdaten: E-Mail-Adresse und Passwort (verschlüsselt) bei der Registrierung via Firebase Authentication.

Spieldaten: Spielergebnisse, Statistiken, Wurf-Koordinaten (Trefferbild), Checkout-Daten. Diese Daten werden ausschließlich zur Darstellung deiner persönlichen Statistiken verwendet.

Gesundheitsdaten (optional): Schlaf, Belastung und Befinden — nur wenn du die entsprechende Funktion aktivierst.

Technische Daten: Keine Cookies, kein Tracking, keine Weitergabe an Werbetreibende.

3. Zweck der Datenverarbeitung

Deine Daten werden ausschließlich verwendet um:

  • Deinen Account zu verwalten
  • Deine persönlichen Dart-Statistiken zu speichern und anzuzeigen
  • Dir KI-Coach-Analysen zu ermöglichen (Daten werden temporär an Anthropic Claude API übermittelt, nicht dauerhaft gespeichert)

4. Speicherung

Alle Daten werden in Firebase (Google LLC, USA) gespeichert. Die Übertragung erfolgt verschlüsselt. Google ist nach EU-US Data Privacy Framework zertifiziert.

5. KI-Analyse

Wenn du die Coach-Analyse oder Video-Analyse nutzt, werden deine Statistikdaten temporär an die Anthropic Claude API übermittelt. Diese Daten werden nicht für das Training von KI-Modellen verwendet und nicht dauerhaft gespeichert.

6. Sprachausgabe

Für die Sprachansagen wird die ElevenLabs API verwendet. Der anzusagende Text wird temporär übermittelt und nicht gespeichert.

7. Deine Rechte

Du hast das Recht auf Auskunft, Berichtigung und Löschung deiner Daten. Zur Löschung deines Accounts wende dich an: hello@darttrainer.app
